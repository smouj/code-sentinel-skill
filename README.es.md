# Code Sentinel

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Code Sentinel para OpenClaw.

## Qué Soluciona
Detecta vulnerabilidades en los commits de código.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque centrado en la seguridad
- Soporte para reversión

## Uso
/code-sentinel

## Ejemplo
```
Analizar código para problemas de seguridad.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
